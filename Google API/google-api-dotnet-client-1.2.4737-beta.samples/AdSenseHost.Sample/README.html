Instructions for the Google .NET Client API – AdSenseHost.Sample

Browse Online

  • Browse Source, or main file Program.cs.

Checkout Instructions

Prerequisites: Install Visual Studio or MonoDevelop, and Mercurial. 

cd [someDirectory] hg clone https://code.google.com/p/google-api-dotnet-client.samples/
        google-api-dotnet-client-samples

Set up Project in Visual Studio

  • Open the GoogleApisSamples.sln with Visual Studio
  • Click on Build > Rebuild Solution
  • Execute the .exe in AdSenseHost.Sample\bin\Debug
  • When prompted insert your client credentials which are generated on the Google APIs Console API Access pane.

Set up Project in MonoDevelop

  • Open the GoogleApisSamples.sln with MonoDevelop
  • Click on Build > Rebuild All
  • Execute the .exe in AdSenseHost.Sample\bin\Debug using mono
  • When prompted insert your client credentials which are generated on the Google APIs Console API Access pane.

Association Sessions

In order to run the association session calls, set AssociationSessionSample.cs as startup object of the project and rebuild.
